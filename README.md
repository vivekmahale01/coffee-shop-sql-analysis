# Coffee Shop Sales & Expansion Analysis

## Project Overview

This project analyzes coffee shop sales data using SQL and provides business insights for expansion decisions.

## Objectives

- Analyze revenue performance
- Find top selling products
- Understand customer behavior
- Compare city-wise performance
- Recommend expansion locations

## Tools Used

- SQL
- MySQL
- GitHub

## Skills Demonstrated

- Joins
- Group By
- Aggregate Functions
- Business Analysis
